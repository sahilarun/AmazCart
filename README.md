# AmazCart
AmazCart - Ecommerce Flutter Source code for Android and iOS
## Clone Version of [AmazCart]
-> https://play.google.com/store/apps/details?id=com.amazcart.app

## App Features
- Featured Sliders
- Top Categories
- New User Zone
- Top Brands
- Flash Deals
- Top Picks Products
- Discounted Products
- Product Recommendations
- Notifications
- Variant Products
- Cart
- Checkout
- Order list
- Cancellations list
- Refunds and Disputes products
- Review Products
- Support system
- Wallet system
- Address Book
- Language support
- Currency support

## Payment Gateways
- Cash on Delivery
- Wallet Payment
- Paypal
- Stripe
- PayStack
- Razorpay (India)
- Bank Payment
- Instamojo
- PayTM (India)
- Midtrans
- JazzCash
- Flutterwave
